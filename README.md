# drawing-game

*作成中*

1人が絵を書いて他のプレイヤーが当てるというゲームです。

WebSocketを使って、フロントはReact,バックエンドはGoを使用。

このリポジトリはフロントのコードです。

[https://takutoaoi.github.io/drawing-game/](https://takutoaoi.github.io/drawing-game/)

## 環境構築

`npm ci`

他はpackage.jsonを参照
